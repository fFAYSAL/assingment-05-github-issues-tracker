1️⃣ What is the difference between var, let, and const? give short answer with normal english

answer: var – Old way to declare a variable. Can be re-declared and updated. Its scope is function-level, not block-level.
        let – Modern variable. Can be updated but cannot be re-declared in the same scope. Its scope is block-level.
        const – Constant variable. Cannot be updated or re-declared. Its scope is also block-level



2️⃣ What is the spread operator (...)?

answer: The spread operator (...) is used to expand or copy elements of an array or object.

3️⃣ What is the difference between map(), filter(), and forEach()?
answer:map() – Creates a new array by transforming each item.
       filter() – Creates a new array with only the items that pass a test.
        forEach() – Runs a function on each item but does not create a new array.


4️⃣ What is an arrow function?
 answer:An arrow function is like a shortcut to write a function—it’s small, quick, and uses => instead of the long function word.



5️⃣ What are template literals?
answer:Template literals are like special quotes (``) that let you put variables and expressions inside a string easily.
       let name = "Faysal";
